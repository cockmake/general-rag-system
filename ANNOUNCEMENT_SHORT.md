# 📢 系统公告

## 🎉 欢迎使用 General RAG System！

### ✨ 核心功能

1. **🤖 多模型支持** - GPT、DeepSeek、千问、Gemini 等主流大模型，一键切换！
   - 推荐使用 **千问系列**（Qwen-Max/Plus）- 稳定快速，效果出色
   - DeepSeek 性价比高，GPT-4 质量最佳

2. **📚 智能 RAG 问答** - 上传文档，AI 秒懂你的知识库！
   - 支持 PDF、TXT、Markdown 等格式
   - 向量语义检索，精准匹配相关内容
   - 引用来源可追溯，回答有据可查

3. **🏢 企业级架构** - 多租户 + 工作空间管理
   - 团队协作，权限可控
   - 多知识库分类管理
   - 完整审计日志

4. **💬 智能对话** - 流式输出 + 上下文记忆
   - 多轮对话，越聊越懂你
   - Markdown 渲染、代码高亮
   - 对话历史永久保存

5. **🔒 隐私优先** - 支持完全本地部署
   - 可选本地向量化（vLLM + embedding_rerank）
   - 私有知识库，数据 100% 自主可控
   - Docker 一键部署

6. **🎨 现代化 UI** - Vue 3 精美界面
   - 深色/浅色主题切换
   - 响应式设计，桌面移动兼容

---

### 💡 使用建议

- **模型选择**: 日常推荐千问 Qwen-Plus，高质量需求选 GPT-4 或 Qwen-Max
- **文档上传**: 按主题分类建立知识库，文档分块建议 500-1000 字符
- **隐私安全**: 敏感文档建议使用本地向量化，定期备份数据
- **遇到错误**: 刷新页面或重试即可
